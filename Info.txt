SONG FOLDER SHOULD BE FORMATED AS SUCH:
song1.jpg
song1.mp3
song2.jpg
song2.mp3
AND SO ON...

NOTES:
1) SMALL LAG WHEN LOADING SONG FOR FIRST TIME
2) EFFECTS MIGHT CAUSE LAG ON OLD DEVICES
3) MUST BE jpg AND mp3 FILES
4) NO OTHER FILES IN SONG FOLDER
5) MAKE SURE THE mp3 FILE GOES RIGHT AFTER THE CORRESPONDING jpg FILE
6) MAKE SURE THERE ARE AN EVEN NUMBER OF mp3 AND jpg FILES
